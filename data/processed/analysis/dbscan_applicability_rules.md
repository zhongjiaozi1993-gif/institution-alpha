# DBSCAN BUY 适用性筛选规则 (v1)

生成时间: 2026-07-09

> 基于 Top100 DBSCAN BUY 信号 A/B/C/D 分组分析生成。
> 目标: 判断哪些股票适合使用 DBSCAN BUY 信号，哪些不适合。
> 这是研究候选规则，不是实盘规则。

---

## 一、适合 DBSCAN BUY 的股票

满足以下所有条件的股票，DBSCAN BUY 信号可能有效：

| # | 条件 | 阈值 | A组实际范围 |
|----|------|------|-------------|
| 1 | n_buy_ops | >= 100 | 989 - 4046 |
| 2 | avg_fwd_5d | > 0 | 1.68% - 7.99% |
| 3 | win_5d | > 52% | 52.5% - 67.5% |
| 4 | avg_universe_excess_fwd_5d | > 0 | 0.43% - 4.26% |
| 5 | stability | > 0 | 0.0510 - 0.3270 |

满足以上 5 条的股票，在 Top100 中出现概率: 13/100（13%）。

### 加分项（建议范围，非必须）

| 维度 | 建议范围 | 依据 |
|------|----------|------|
| 聚类委托数 | 26 ~ 50 | A组25-75分位 |
| 聚类时长(min) | 23.1 ~ 28.8 | A组25-75分位 |
| VWAP偏离(%) | -0.070 ~ 0.000 | A组25-75分位 |
| 单笔BUY均额(万) | 1018.3 ~ 2396.8 | A组25-75分位 |

## 二、不适合 DBSCAN BUY 的股票

以下任一条件触发的股票，DBSCAN BUY 信号大概率无效：

| 类型 | 特征 | 在 Top100 中占比 |
| D: 无效 | avg_fwd_5d <= 0，信号完全没有预测能力 | 44只（44%） |
| B: 纯Beta | avg_fwd_5d > 0 但 universe_excess <= 0，收益来自市场而非Alpha | 35只（35%） |
| C: 弱信号 | avg_fwd_5d > 0 但 win_5d <= 52%，信号不够稳健 | 8只（8%） |

**B+C+D 合计 87 只（87%）。**
在这些股票上使用 DBSCAN BUY 信号，预期收益为负或无超额。

## 三、不适用信号的特征总结

如果一只股票的 DBSCAN BUY 信号呈现以下特征，应排除：

1. **BUY 信号产生正向绝对收益但无超额收益** → 信号捕获的是市场Beta，无选股价值
2. **win_5d < 50%** → 硬币都不如，随机买入都比信号好
3. **stability < 0** → 超额收益不稳定，时好时坏，无法依赖
4. **BUY ops 很多但 universe_excess 为负** → 信号量大不代表质量好

## 四、应用方法

1. 对任意候选股票池，先筛选满足"适合条件"1-5的股票
2. 加分项作为 tie-breaker，不满足不排除
3. 筛选后的候选池需经 OOS 验证方可实盘
4. 当前规则仅适用于 V6 DBSCAN BUY 信号，不适用于 SELL/其他信号类型
