## 规则来源

A组(13只) vs 其他组(87只) 在各维度上的差异，取 A 组 25-75 分位作为建议阈值。


## 一级规则（必须满足）

| 条件 | 阈值 | 依据 |
|------|------|------|
| n_buy_ops | >= 989 | A组最小值，确保信号量充足 |
| win_5d | > 52% | 扩容标准，A组均值 59.5% |
| avg_universe_excess_fwd_5d | > 0 | 必须有正向选股能力，A组均值 1.14% |
| stability | > 0 | 收益稳定性为正，A组均值 0.1102 |
| avg_fwd_5d | > 0 | 绝对收益为正，A组均值 3.18% |

## 规则应用说明

1. **一级规则**全部满足 → 进入 production candidate 候选池

2. **二级规则**作为参考，不满足不排除，但需标记为"需人工审核"

3. 当前规则基于 2025 年全年样本内数据，需 2026 H1 OOS 验证

4. 规则仅适用于 DBSCAN BUY 信号（v6 流水线），不适用于其他信号类型

5. 在 Top100 池中，一级规则筛选出 13 只（13%）


## 已知无效的信号特征

以下特征的股票在 Top100 中 DBSCAN BUY 信号无效:

- **D 组(44只)**: avg_fwd_5d <= 0，信号完全没有预测能力

- **C 组(35只)**: avg_fwd_5d > 0 但 universe_excess <= 0，收益来自市场beta而非alpha

- **B 组(8只)**: 正收益但不满足全部条件，信号不够稳健


**C+D 组共 79 只（79%），扩容的主要噪音来源。**
