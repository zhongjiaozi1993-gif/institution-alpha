## 问题 1：Top100 是否通过？

### 扩容标准检查

| 标准 | 阈值 | 实际 | 结果 |
|------|------|------|------|
| fwd_5d > 0 | > 0 | +0.53% | 通过 |
| win_5d > 52% | > 52% | 48.7% | **未通过** |
| universe_excess_fwd_5d > 0 | > 0 | -0.83% | **未通过** |
| 月度 fwd_5d ≥ 8/12 为正 | ≥ 8 | 9 | 通过 |
| Top 5 贡献 < 50% | < 50% | 33.3% | 通过 |
| 价格缺失 < 5% | < 5% | 0% | 通过 |
| 前向收益缺失 < 10% | < 10% | 2.5% | 通过 |

**4/7 通过。Top100 未通过扩容标准。**

### 未通过原因分析

**1. win_5d = 48.7%（阈值 52%）**

DBSCAN BUY 信号在 100 只股票上的平均胜率低于 50%。这意味着：
- 随机选一个 DBSCAN BUY 信号，赚钱概率不到一半
- 扩容稀释了信号质量：Pri25 的 58.7% → Top100 的 48.7%
- 100 只中仅 21 只 universe_excess_fwd_5d > 0

**2. universe_excess_fwd_5d = -0.83%（阈值 > 0）**

DBSCAN BUY 信号选出的股票，其后续收益**低于**同日期 Universe 全部股票的等权平均。这意味着：
- 信号没有选股能力——它选的股票跑输同池平均水平
- 信号的正绝对收益(+0.53%)完全来自市场 beta（Universe 同期 +1.36%）
- DBSCAN 的聚类识别的是"当日活跃股票"而非"即将上涨的股票"

### 失败股票特征

从 Top100 中剔除 Pri25 重叠的 19 只后，剩余的 81 只"新增"股票：

- 平均 win_5d: ~46%
- 平均 universe_excess_fwd_5d: 显著为负
- 这些股票流动性更高（Top100 按成交额排序），但 DBSCAN 信号质量更差

**大成交额股票 ≠ DBSCAN 信号更有效。** 反而可能更差——大盘股中的"机构行为"更可能是正常的市场噪音而非 alpha 信号。

## 问题 2：是否可以扩到 Top200？

**不建议。**

理由：
1. Top100 已未通过核心指标（win_5d、universe_excess），Top200 会更差
2. 信号质量从 Pri25 → Top100 已显著下降，继续扩容只会进一步稀释
3. 300 只全量历史数据（win_5d=46.1%）已证明更大池子无 alpha
4. 13 只 production candidates 集中在 Pri25 和邻近股票，没有证据表明更广的池子中有隐藏的优质信号

### 当前有效范围

```
Pri25 精选 (25只)  → win_5d=58.7% ✓
Production Candidates (13只) → 都满足 win_5d>52%, excess>0 ✓
Top100 (100只) → win_5d=48.7% ✗
All 300 (300只) → win_5d=46.1% ✗
```

**V6 DBSCAN BUY 信号的有效股票池约 15-25 只。** 不建议无差别批量扩容。

## 当前最大风险

### 1. 幸存者偏差（高）
这 13 只 production candidates 是从 300 只中筛选出来的。可能存在数据挖掘偏差。

### 2. 样本内评估（高）
全部分析基于 2025 年全年数据，无训练/测试分离。需 2026 年 OOS 验证。

### 3. 信号缺乏超额收益（高）
Top100 的 universe_excess 为负，说明信号选股能力弱。Production candidates 的 universe_excess 为正可能只是统计波动。

### 4. 行情阶段依赖（中）
H2 信号量和质量均优于 H1。如果市场环境变化（如进入熊市），信号可能全面失效。

### 5. Benchmark 口径（中）
当前 index_bench 字段预留但未接入实际指数数据。Universe benchmark 虽比 signal benchmark 更有意义，但仍非市场基准。

### 6. entry_price 收益未计算（低）
因 hfq/qfq 价格坐标不匹配，entry_price 收益暂不可用。不影响 close-to-close 评估结论。
