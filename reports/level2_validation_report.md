## 4. 样本外(OOS)增量（train 选参 / test 固定评估）

train ≤ 2025-08-31（155 日）选出综合分特征 **l2_late_net_ratio, l2_big_net_ratio, l2_super_net_yi, l2_intraday_ret, l2_large_share, l2_vwap_close_dev**，最优 alpha=**signal033**，权重 w_a=0.1024/w_l=0.0261；test ≥ 2025-09-01（141 日）固定评估。

> **口径**：alpha 与 L2 综合分都按 **train 端方向**定向后再评估（committed direction，OOS 不允许用 test 端符号），融合 = w_a·z(alpha_定向)+w_l·z(L2_定向)。表中 Alpha191 = train-定向后的 RankIC。

> 5d 参照 alpha=**signal033**：train 方向 sign_a=-1，test 上 train-定向 RankIC=-0.0523 / 未定向 raw=0.0523　→ **方向在 OOS 反转**（train 上有效的符号到 test 失效）。

> OOS 增量判定：**名义增量** = 融合方向泛化(RankIC>0) 且 定向融合 RankIC > 定向 Alpha191；**稳健增量** = 名义增量 且 **同 horizon** L2 综合分自身方向泛化(避免跨 horizon 拼凑)。
> 名义增量 horizon: **10d**；L2 综合分方向泛化 horizon: **5d**；两者**同 horizon 同时成立（稳健增量）**: **无**。
> ⚠️ 名义增量与 L2 方向泛化落在**不同 horizon**：5d 上 L2 综合分泛化但融合无增量；10d 上融合有名义增量，但那是**赢过一个已塌缩(方向反转后近于零)的定向 alpha**、且同 horizon L2 综合分方向未泛化。**不构成稳健 OOS 增量。**

## 5. 分级结论

| 层级 | 结论 |
|---|---|
| 样本内增量 | 部分增量 |
| OOS 名义增量（定向融合 > 定向 Alpha191） | 有（10d） |
| L2 综合分 OOS 方向泛化 | 是（5d test RankIC>0） |
| **OOS 稳健增量（同 horizon 名义增量∧L2泛化）** | **无** |
| 参照 Alpha191 OOS 方向稳定性 | 方向反转（5d/10d，train 符号到 test 失效） |
| **是否建议进入 Phase 6 规则融合** | **暂不建议（先纯 Alpha191 融合）** |

> 注意：样本内最优参照 Alpha191 在 OOS（5d/10d）方向反转——在 train 上有效的符号到 test 已失效。这正是 OOS 要暴露的“样本内选优”陷阱：样本内增量高度依赖那个不稳定的 alpha，不可直接外推。

> 建议：OOS 仅有**名义增量而无同 horizon 稳健增量**（名义增量落在 L2 未泛化的 horizon、且赢过的是已塌缩的定向 alpha），**暂不进入** Phase 6 的 Level-2 融合。Phase 6 先做**纯 Alpha191 规则融合**；Level-2 待扩面板（更多股票/更密日历）或 Phase 9 ML+滚动 OOS 再评估。
