## 5. 分级结论

| 层级 | 结论 |
|---|---|
| 样本内增量 | 有增量 |
| OOS 名义增量（定向融合 > 定向 Alpha191） | 有（10d） |
| L2 综合分 OOS 方向泛化 | 是（5d/10d test RankIC>0） |
| **OOS 稳健增量（同 horizon 名义增量∧L2泛化）** | **有（10d）** |
| 参照 Alpha191 OOS 方向稳定性 | 未见反转 |
| **是否建议进入 Phase 6 规则融合** | **建议（小权重增强项）** |

> **增量归因（重要）**：该 OOS 增量主要来自**规模 / 成交活跃度 / 流动性反转**代理（成交额、大单金额等量级特征负 IC 最强），**不是**已验证的**机构净买入 / 拆单方向** edge——净流入、早尾盘净流、聚类净买入等方向变量 |RankIC_5d|≈0。且 L2 综合分与 signal019/040 日均 Spearman≈0.77，信息高度重叠。故“OOS 稳健增量：有（10d）”保留，但**不能视作独立机构行为 Alpha**；是否只是已有规模因子的重复表达，待 Phase 5.2C 中性化检验后再定性。

> 建议：Level-2 综合分在 OOS（10d）上同 horizon 既方向泛化又对定向 Alpha191 有增量，可进入 Phase 6，将 Level-2 综合分作为 Alpha191 排序的**小权重增强项**（先小仓验证，避免因面板窄而过配）。
